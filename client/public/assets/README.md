# Logo Assets

Place the Dubai Tango Festival logo image here.

**Required file:**
- `logo.png` - The main logo image

The logo should be a PNG file with a transparent background (recommended) or white background. The component will automatically scale it to the appropriate size.

**Current usage:**
- The logo is used across all pages of the website:
  - Registration page (header and footer)
  - Workshops page (header)
  - Admin page (sidebar and header)
  - Admin login page
  - Confirmation page

**Image specifications:**
- Format: PNG (recommended) or JPG
- Recommended dimensions: At least 200px width for best quality
- Aspect ratio: Maintain original proportions
- Background: Transparent or white
